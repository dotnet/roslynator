# RCS0025: Add newline before accessor of full property

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0025    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
string Foo { get { return _foo; } set { _foo = value; } }
```

### Code with Fix

```csharp
string Foo
{
    get { return _foo; }
    set { _foo = value; }
}
```
