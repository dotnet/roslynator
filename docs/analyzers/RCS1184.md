# RCS1184: Format conditional expression \(format ? and : on next line\)

| Property | Value      |
| -------- | ---------- |
| Id       | RCS1184    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
var y = (x) ? // RCS1184
Foo() :
Bar();
```

### Code with Fix

```csharp
var y = (x)
? Foo()
: Bar();
```
