# RCS0018: Add newline after conditional operator instead of before it

| Property | Value      |
| -------- | ---------- |
| Id       | RCS0018    |
| Category | Formatting |
| Severity | None       |

## Example

### Code with Diagnostic

```csharp
var y = (x)
    ? Foo() // RCS0018
    : Bar();
```

### Code with Fix

```csharp
var y = (x) ?
     Foo() :
     Bar();
```
