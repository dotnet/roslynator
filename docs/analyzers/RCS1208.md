# RCS1208: Reduce if nesting

| Property                    | Value   |
| --------------------------- | ------- |
| Id                          | RCS1208 |
| Category                    | Style   |
| Default Severity            | Info    |
| Enabled by Default          | \-      |
| Supports Fade\-Out          | \-      |
| Supports Fade\-Out Analyzer | \-      |

## Example

### Code with Diagnostic

```csharp
if (condition1) // RCS1208
{
    Foo1();

    if (condition2)
    {
        Foo2();

        if (condition3)
        {
            Foo3();
        }
    }
}
```

### Code with Fix

```csharp
if (!condition1)
{
    return;
}

Foo1();

if (!condition2)
{
    return;
}

Foo2();

if (!condition3)
{
    return;
}

Foo3();
```

## How to Suppress

### SuppressMessageAttribute

```csharp
[assembly: SuppressMessage("Style", "RCS1208:Reduce if nesting.", Justification = "<Pending>")]
```

### \#pragma

```csharp
#pragma warning disable RCS1208 // Reduce if nesting.
#pragma warning restore RCS1208 // Reduce if nesting.
```
